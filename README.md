# aliyun-esdk

`aliyun-esdk` 是阿里云SDK的 Erlang 实现，包括一些基础API。

## 项目说明

- [public-parameters](https://github.com/aliyun-esdk/public-parameters)
  
  阿里云 [公共参数](https://help.aliyun.com/document_detail/29745.html) 计算，也是所有阿里云API调用的基石。
